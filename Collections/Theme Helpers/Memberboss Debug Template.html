Memberboss Debug Template

Features displayed on this page are installed by the Memberboss Features collection. If the templates that include these elements are uploaded with the HtmlImport tool, they can be added:

To add to a template that will be uploaded with the HtmlImport tool, use the following.

Add to a template that will be uploaded with the Html Import tool.

          <nav class="navbar navbar-expand-lg header-block-top">
            <div class="collapse navbar-collapse" id="navbar" data-addon="Template-Header-Navbar-Nav" >
              <!-- placeholder html that is removed when template is uploaded -->
            </div>
          </nav>
        
Add to a template that will be added manually, or within an Add-on collection.

          <nav class="navbar navbar-expand-lg header-block-top">
            <div class="collapse navbar-collapse" id="navbar">
              {% Template-Header-Navbar-Nav %}
            </div>
          </nav>

Template-Header-Navbar-Nav

Executes the Navbar-Nav addon for the template header. The navbar-nav addon creates html compatible with the Bootstrap navbar in the position where the class="navbar-nav" exists.

How to Use

            <nav class="navbar navbar-expand-lg header-block-top">
              <div class="collapse navbar-collapse" id="navbar" data-addon="Template-Header-Navbar-Nav" >
                <!-- placeholder html that is removed when template is uploaded -->
              </div>
            </nav>
Example
{% Template-Header-Navbar-Nav %}

Template-Header-Brand-Img

Is replaced with an html image tag. It should be used in the navbar brand of the header section in the template.

How to Use

            <nav class="navbar navbar-expand-lg navbar-light bg-light">
              <a class="navbar-brand" href="#" data-addon="Template-Header-Brand-Img"><img href="placeholder.png"></a>
            </nav>
